# rGUIDANCE
## Multiple sequence alignment reliability scores

You can install this R package by using devtools as follows:

```r
install.packages("devtools")
devtools::install_github("FranzKrah/rGUIDANCE")
```
and then use it by typing ...

```r
library("rGUIDANCE")
```
